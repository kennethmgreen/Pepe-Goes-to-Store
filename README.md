## Pepe-Goes-to-Store
This game centers on your quest to acquire chips. You’ll explore various locations, each leading to different possible endings. Manage your money carefully so you can afford the chips when the time comes. You begin the game with 0 dollars.


## How to run the game

1. Download or clone the repository  
2. Make sure Python is installed  
3.Run the game with Python 3

4. Important: When the game asks you to make a choice, type the full name of your choice exactly as shown.

Example: If the game asks “Where would you like to go?” and shows options:

  1. Stay home
  2. Somewhere random
  3. Space
  4. Store

You must type:
Stay home


## Tips
Keep track of your money — some paths will require you to have enough to buy chips.

Experiment with different locations to see all possible endings.

Pay attention to the wording of prompts — the game reads exactly what you type.
